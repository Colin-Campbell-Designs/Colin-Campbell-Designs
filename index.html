css3

HTML5 and CSS3 are the building blocks of the web, ever website starts with these 2 technologies. HTML is the content in every website, text, images etc, CSS3 is what makes everything look a feel good about the website, colours, layout etc.

javscript

JavaScript is what makes our website comes to life. Its the technology that make everything interactive. Have you ever clicked a button on site and something changed or scrolled down a web page and an image appeared? Thats javaScript, its the most popular programming language on the web

UX

UX stands for user experience and this is the process of designing a website with the user in mind. Ux is the is the first step in deploying your website online from it being just an idea in your head. We take your idea a begin designing your site keeping in mind the best possible experience for the end user.

RESPONSIVE DESIGN

Responsive design is when your website changes from when its viewed on a desktop screen which can be quite large, to a mobile device which is a lot smaller with loosing any important content and also still providing the same user experience

seo

SEO stands for search engine optimisation and this is whats needed to list your website on google or another search engine. Not only do you want your site to be seen you also want it to rank well on a search engine making it easier for your userâ€™s to find your site.

react js

React JS is a javascript framework that helps us a developers to build usable components for your websites user interface. It not only makes it faster to build websites but it also makes the website run a lot faster, which in turn provides a fantastic user experience
